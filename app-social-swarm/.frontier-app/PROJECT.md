# Social Swarm — Content & Social Media Marketing

## What This Is

An agentic swarm app for Frontier OS that automates content creation and social media marketing for Frontier Tower. A user defines a campaign brief (goal, tone, target audience, platforms), and the swarm — a coordinated team of specialised AI agents — generates platform-native content: tweets, LinkedIn posts, Instagram captions, and Farcaster casts, complete with image generation prompts and hashtag sets.

Each agent in the swarm has a defined role (copywriter, image prompter, hashtag optimizer, tone adapter, content planner) and is paid per run using Frontier Dollars (FND) via the x402 HTTP payment protocol.

## Core Value

One brief → a full content calendar across every major social platform, all generated by a coordinated agent swarm and paid for on-chain in FND.

## SDK Modules

| Module | Why Needed | Key APIs |
|--------|-----------|----------|
| Wallet | FND payments for each swarm agent run | `getBalanceFormatted()`, `transferOverallFrontierDollar()`, `getAddress()` |
| User | Display user identity, gate campaign creation to members | `getDetails()`, `getProfile()` |
| Storage | Persist campaigns, generated content, payment history | `get()`, `set()`, `remove()` |

## Target Users

- **Primary:** Frontier Tower members who want to produce social media content without a dedicated marketing team
- **Secondary:** Community managers running campaigns for Frontier Tower events and announcements
- **Non-users:** Anonymous visitors (app requires Frontier Wallet)

## Constraints

- **Runtime:** Runs inside Frontier OS iframe
- **Theme:** Dark theme mandatory
- **SDK:** All platform features via `useServices()` abstraction
- **Auth:** Identity provided by Frontier OS
- **Standalone:** Mock services provide realistic swarm simulation during development
- **x402:** Each swarm agent charges FND per campaign run; payments are processed via `transferOverallFrontierDollar`
